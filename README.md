# صانع ريلز دلائل الخيرات (Web)

## ماذا يفعل؟
- يجلب نص الأحزاب 1..8 تلقائيًا من مصدر: dalailalkhayrat.com (Server-side proxy عبر API route)
- يسمح باختيار نطاق جمل متتابع (Range)
- معاينة حيّة 9:16 (خلفية لون/صورة + إعدادات نص + إظهار/إخفاء رقم الحزب واسم القارئ)
- تشغيل صوت مقصوص تلقائيًا (يتطلب mp3 + توقيتات)
- صفحة Admin لإنشاء التوقيتات JSON لكل (قارئ/حزب) مع Start/End

> المصدر: https://www.dalailalkhayrat.com/parts.php?part=1  (وباقي الأحزاب بتغيير الرقم)

## التشغيل محليًا
1) ثبّت Node.js (يفضل 18+)
2) داخل مجلد المشروع:
```bash
npm install
npm run dev
```
ثم افتح: http://localhost:3000

## إضافة ملفات الصوت لاحقًا
- افتح: `data/readers.json`
- ضع روابط mp3 لكل حزب داخل `audioParts`:
```json
"audioParts": {
  "1": "https://.../part-1.mp3",
  ...
  "8": "https://.../part-8.mp3"
}
```
> مهم: روابط mp3 يجب أن تسمح بالتحميل عبر المتصفح (CORS)، خصوصًا للتصدير عبر ffmpeg.wasm.

## إنشاء التوقيتات
- افتح: `/admin/timings`
- اختر القارئ والحزب
- ضع رابط mp3
- شغّل الصوت وحدد Start/End لكل جملة
- احفظ (localStorage) أو نزّل JSON

## الترجمة
- الترجمة اختيارية (Toggle)
- يتم جلبها تلقائيًا من نفس صفحة المصدر **إن كانت موجودة** لكل مقطع (فرنسية/إنجليزية) عبر heuristics.
- إذا لم توجد ترجمة للمقطع، يظهر تنبيه داخل المعاينة.
